# [프로젝트 1] 구글 폼 응답 자동 분류 시스템 구축

## 1. 프로젝트 개요

구글 폼을 통해 수집되는 응답 데이터를 수동으로 정리하는 번거로움을 해결하고, 이메일 도메인(Gmail vs 기타)에 따라 데이터를 자동으로 분류하여 관리하는 자동화 시스템을 구축함.

## 2. 사용 도구

| **비교 항목** | **도구 A (Make)** | **도구 B (Zapier)** |
| --- | --- | --- |
| **난이도** | 중 (비주얼 로직 구성 필요) | 하 (직관적인 1:1 매핑) |
| **자유도** | 매우 높음 (필터/라우터 무제한) | 보통 (조건 설정에 제한 있음) |
| **비용 효율** | 높음 (복잡한 작업 시 유리) | 낮음 (기능 추가 시 비용 상승) |
| **적합한 상황** | 복잡한 비즈니스 프로세스 자동화 | 단순 반복 업무 빠르게 연결할 때 |

## 3. 구현 로직 (Make Scenario)

데이터는 아래 순서로 처리됨:

1. **Watch New Rows (Trigger):** 구글 시트에 새로운 응답이 추가되는 것을 실시간 감지.
2. **Router (분기):** 이메일 도메인 조건을 바탕으로 데이터를 두 경로로 분류.
    - **경로 A (Gmail):** 필터 조건 `Contains 'gmail.com'`
    - **경로 B (기타 도메인):** 필터 조건 `Does not contain 'gmail.com'`
3. **Add a Row (Action):** 분류된 경로에 따라 각각 지정된 시트 탭에 데이터 자동 기입.

## 4. 핵심 설정 내역

- **필터링 전략:** 라우터의 각 경로에 필터를 적용하여 데이터가 중복되거나 잘못 분류되는 현상을 방지함.
- **중복 방지 설정:** `Watch Rows` 모듈의 'Choose where to start'를 `From now on`으로 설정하여 중복 응답 수집 방지.
- **데이터 매핑:** 구글 폼 응답 데이터(성함, 이메일, 피부 고민 등)를 시트의 각 열에 1:1 대응하여 매핑.

## 5. 기대 효과

- **업무 효율성 증대:** 응답 수집 후 수동 분류 과정(0분)으로 단축.
- **휴먼 에러 방지:** 도메인별 자동 분류를 통해 데이터 관리 실수 0% 구현.
- **확장성 확보:** 추후 자동 메일 발송, 알림 연동 등 프로젝트 2로 전환 가능한 구조 마련.

# [프로젝트 2] 체험단 신청 자동 응답 파이프라인 고도화

## 1. 프로젝트 목적

- 신청자에게 즉각적인 접수 확인 메일을 발송하여 **브랜드 신뢰도 확보** 및 운영자의 **수동 안내 업무 0% 달성**.
- 시스템의 데이터 분류(Project 1)와 연동하여 **데이터 관리와 커뮤니케이션의 일원화**.
- **선정 도구:** **Make**
- **선정 이유:**
    1. **시각적 워크플로우 설계:** 라우터(Router)와 필터 기능을 통해 데이터의 흐름을 직관적으로 시각화할 수 있어, 복잡한 조건 분기가 필요한 비즈니스 로직을 설계하기에 최적입니다.
    2. **데이터 매핑의 정교함:** 구글 폼에서 넘어오는 개별 데이터(이름, 이메일, 피부 고민 등)를 메일 모듈에 직접 매핑하는 과정에서 타 도구 대비 훨씬 유연하고 세밀한 설정이 가능합니다.
    3. **확장성 및 유지보수:** 향후 신청 인원이 대규모로 증가하거나, 도메인별 마케팅 전략이 고도화될 때 구조를 변경하지 않고도 모듈 단위의 수정만으로 대응이 가능한 시스템 구조를 구축할 수 있습니다.

## 2. 구현 로직 (Make Scenario 업데이트)

1. **Trigger (Watch Rows):** 구글 폼 응답 발생 감지.
2. **Action 1 (Gmail - Send an email):** 신청자 개인별 맞춤형 '접수 확인 메일' 즉시 발송.
    - *핵심 전략: '스팸 방지 표준 템플릿'을 적용하여 높은 도달률 보장.*
3. **Router (조건 분기):** 수신된 데이터를 도메인별(Gmail vs 기타)로 분류.
4. **Action 2 & 3 (Add a Row):** 분류된 데이터를 시트별 탭에 자동으로 적재

## 3. 핵심 적용 기술

- **개인화 데이터 매핑:** `{{성함}}`, `{{피부고민}}` 등 구글 폼의 변수를 메일 본문에 자동 매핑하여 스팸성 대량 발송이 아닌 '1:1 소통 메일'로 인식되도록 설계함.
- **스팸 방지 메커니즘:** 메일 하단에 '접수 확인 자동 안내 문구'를 삽입하고, 제목과 본문에 광고성 키워드를 배제하여 메일 서버 평판 유지.

## 4. 기대 효과

- **운영 효율:** 지원자별 수동 메일 발송 업무가 시스템 자동화로 대체됨.
- **고객 경험(CX):** 즉각적인 반응을 통해 지원자의 브랜드 충성도 증대.
- **데이터 관리:** 분류된 데이터를 바탕으로 추후 도메인별 타겟팅 마케팅 전략 수립 가능.
